# Design

All design and build files for the marjutrimminen.com site live here.

The current working version is **`v1/`**. When the design changes substantially enough to be a new direction, copy `v1` to `v2` rather than overwriting in place — this keeps prior states browsable and comparable. Update the repository-root `index.html` redirect to point at the latest version when promoting a new working version.

For the current site's structure, typographic system, and per-section decisions, see `v1/_readme.md`.

For visual asset provenance, see `v1/images/_readme.md` and `../Imagery/_readme.md`.

For project-level handover notes (deployment, source archives, contact details, missing pieces), see `../_readme.md`.
